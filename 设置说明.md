# OlmoEarth 最简化目录结构设置说明

## 目标目录结构

```
olmoearth/                          # 新建文件夹
├── olmoearth_inference_jp2.py      # 推理脚本（已创建）
├── config.py                        # ⭐ 从olmoearth_pretrain复制
├── datatypes.py                     # ⭐ 从olmoearth_pretrain复制
├── nn/                             # ⭐ 从olmoearth_pretrain/nn复制
│   ├── __init__.py                 # ⭐ 新建
│   ├── galileo.py                  # 主模型架构
│   ├── flexi_vit.py                # Encoder/Decoder
│   ├── attention.py                 # 注意力机制
│   ├── flexi_patch_embed.py        # Patch嵌入
│   ├── encodings.py                 # 位置编码
│   ├── pooling.py                   # 池化
│   └── utils.py                     # 工具函数
├── data/                           # ⭐ 从olmoearth_pretrain/data复制
│   ├── __init__.py                 # ⭐ 新建
│   ├── constants.py                 # 模态定义
│   └── normalize.py                 # 归一化
└── params/                         # ⭐ 从HuggingFace下载
    ├── config.json                  # 模型配置
    └── weights.pth                  # 模型权重
```

## 需要复制的文件清单

### 1. 根目录文件 (2个)
- `config.py`
- `datatypes.py`

### 2. nn目录文件 (7个)
- `nn/__init__.py` (新建)
- `nn/galileo.py`
- `nn/flexi_vit.py`
- `nn/attention.py`
- `nn/flexi_patch_embed.py`
- `nn/encodings.py`
- `nn/pooling.py`
- `nn/utils.py`

### 3. data目录文件 (3个)
- `data/__init__.py` (新建)
- `data/constants.py`
- `data/normalize.py`

## 复制命令示例

```bash
# 1. 创建目录结构
mkdir -p olmoearth/nn olmoearth/data olmoearth/params

# 2. 复制根目录文件
cp olmoearth_pretrain-main/olmoearth_pretrain/config.py olmoearth/
cp olmoearth_pretrain-main/olmoearth_pretrain/datatypes.py olmoearth/

# 3. 复制nn目录文件
cp olmoearth_pretrain-main/olmoearth_pretrain/nn/galileo.py olmoearth/nn/
cp olmoearth_pretrain-main/olmoearth_pretrain/nn/flexi_vit.py olmoearth/nn/
cp olmoearth_pretrain-main/olmoearth_pretrain/nn/attention.py olmoearth/nn/
cp olmoearth_pretrain-main/olmoearth_pretrain/nn/flexi_patch_embed.py olmoearth/nn/
cp olmoearth_pretrain-main/olmoearth_pretrain/nn/encodings.py olmoearth/nn/
cp olmoearth_pretrain-main/olmoearth_pretrain/nn/pooling.py olmoearth/nn/
cp olmoearth_pretrain-main/olmoearth_pretrain/nn/utils.py olmoearth/nn/

# 4. 复制data目录文件
cp olmoearth_pretrain-main/olmoearth_pretrain/data/constants.py olmoearth/data/
cp olmoearth_pretrain-main/olmoearth_pretrain/data/normalize.py olmoearth/data/

# 5. 创建__init__.py文件
echo '"""NN modules"""' > olmoearth/nn/__init__.py
echo '"""Data modules"""' > olmoearth/data/__init__.py

# 6. 下载模型到params目录
# 从 HuggingFace: https://huggingface.co/allenai/OlmoEarth-v1-Base
# 下载 config.json 和 weights.pth 到 olmoearth/params/
```
